# my dotfiles

## Requirements

- zsh
- curl
- wget
- git
- stow
- tmux
- unzip
- subversion
- lua5.3
- [nvim](https://github.com/glepnir/jarvim)

```
./init.sh init && ./init.sh preinstall
```

## 安装

```shell
stow zsh # 默认安装到 $HOME 目录
```

## 卸载

```shell
stow -D zsh # 卸载 zsh
```
