# my dotfiles

## Requirements

COMMANDS=(zsh stow git curl unzip bat exa fzf fd rg)
- zsh
- stow
- git
- curl
- unzip

```bash
./init
```

## Init Submodule
```bash
git submodule init # init the submodule
git submodule update
```

## Install
```bash
stow vim zsh # etc...
```

## Uninstall
```bash
stow -D vim
```
